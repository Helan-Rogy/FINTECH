# Phase 7 - Accessibility Layer MVP

## Objective
Improve usability for low-literacy and first-time users.

## Tasks
- Add chatbot-style assistant screen in web app.
- Implement FAQ + report explanation prompts.
- Provide simplified language mode for reports.
- Add optional voice placeholder integration point:
  - UI toggle + API contract stub

## Deliverables
- Accessibility UI module
- Prompt/response templates for simple explanations
- `docs/accessibility-guidelines.md`

## Exit Criteria
- User can ask basic questions and receive understandable responses.
- Report can be rendered in normal and simplified language versions.
