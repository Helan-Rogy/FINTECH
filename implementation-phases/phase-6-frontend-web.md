# Phase 6 - Frontend Web App

## Objective
Deliver the MVP user and analyst experience in React.

## Tasks
- Set up React + Tailwind frontend.
- Build user screens:
  - Data upload/select sample
  - Fraud/risk result cards
  - Explanation report view
- Build analyst screens:
  - Flagged transactions table
  - Case detail and decision action
  - Audit trail/history
- Integrate with backend APIs and auth.

## Deliverables
- Working frontend app with route-based role views
- API integration and error/loading states
- Basic responsive UI

## Exit Criteria
- User can complete full happy-path flow in browser.
- Analyst can review and close a case.
