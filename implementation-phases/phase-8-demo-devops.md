# Phase 8 - Demo and DevOps

## Objective
Package and present a reliable working prototype.

## Tasks
- Dockerize backend, frontend, and ML service.
- Add seed scripts for sample users/data/cases.
- Create one-command startup (`docker compose up`).
- Add demo script covering:
  - fraud detection scenario
  - risk scoring scenario
  - explanation/report scenario
  - analyst case decision scenario
- Add basic monitoring/logging hooks.

## Deliverables
- `docker-compose.yml`
- Demo runbook (`docs/demo-runbook.md`)
- Seed scripts and environment template

## Exit Criteria
- Entire stack starts locally with documented steps.
- Demo can be executed end-to-end in under 10 minutes.
